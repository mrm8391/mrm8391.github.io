Elastic Surface Fracture

By Maximillian McMullen and Tristan Manning

The goal of this project is to simulate the stretchy properties of rubber. A simple object is pressed into a rubber sheet, with the sheet conforming, and eventually tearing from the strain.

Project Overview

Discuss basics of project physics and algorithms here With elastic surfaces, when an object pushes into it, that surface stretches out and conforms to the shape of the object. If the object keeps pushing into the surface, the surface will eventually begin to rip. The surface physics can be simplified to a mesh of spring mass pairs. To tear the surface, springs that reach their respective maximum lengths are removed from the particles' force dependencies. With a perfect grid mesh, these springs will rip in a relatively square pattern, which looks quite unrealistic. This gives motivation to use a less uniform pattern for the mesh, i.e. a Delaunay Triangulation. Inputs: The user of this simulation can experiment with different parameters. For the Delaunay Triangulation itself, the user may adjust the density of the plane tessellation. In other words, the higher the number, the more points that will be triangulated. This is set to a default 50. Existing Work

The spring-mass-damper model is used to represent the elastic properties of the plane. This is commonly used in animations to replicate cloth physics. Pixar has an online course detailing how this is done. There are other ways of modeling non-linearly elastic properties as well; this site discusses several different ways of implementing them. Lastly, while we use simplified collision detection, realistic object collision and deformation can be modeled with the recently developed BD-Trees.
